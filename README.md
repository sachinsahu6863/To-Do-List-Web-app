# ✅ Flow Tasks — Simple To-Do List App

A clean, responsive, and intuitive **To-Do List application** built with **HTML**, **CSS**, and **vanilla JavaScript**.  
Stay organized, track your active tasks, and enjoy smooth filtering — all with persistent local storage.

## ✨ Features

- ✅ **Add new tasks** — quick input with Enter key support.
- ✅ **Mark tasks as complete** — click on any task to toggle completion.
- ✅ **Delete individual tasks** — remove tasks you no longer need.
- ✅ **Filter tasks** — view **All**, **Active**, or **Completed** tasks with one click.
- ✅ **Task counter** — shows remaining (active) tasks at a glance.
- ✅ **Clear completed** — remove all finished tasks in bulk.
- ✅ **Persistent storage** — tasks are saved in your browser's `localStorage`. Refreshing the page won't lose your data.
- ✅ **Responsive design** — works flawlessly on desktops, tablets, and mobile devices.
- ✅ **Smooth animations** — subtle fade-in effects for better UX.
- ✅ **Clean UI** — modern card layout with soft gradients and shadows.

---

## 🛠️ Technologies Used

- **HTML5** — semantic structure.
- **CSS3** — custom styling, flexbox, grid, animations, responsive media queries.
- **JavaScript (ES6+)** — DOM manipulation, event handling, local storage, dynamic rendering.
